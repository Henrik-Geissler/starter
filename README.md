## Getting Started

```bash
# Install dependencies
npm install

# Build all packages
npm run build

# Run Storybook
npm run storybook
```

## Components

- `lux-button` - Button with variants (primary, secondary, outline, ghost)
- `lux-text` - Text with size, weight, and color options
- `lux-heading` - Semantic headings (levels 1-6)
- `lux-card` - Card container with header/footer slots
- `lux-tabs` / `lux-tab` - Tabbed interface
- `lux-scope` - Configuration scoping (density, color)
- `lux-data-table` - Data table (ag-Grid peer dependency)
- `lux-icon` - Icons (Font Awesome peer dependency)

## Usage Example

```html
<lux-scope density="comfortable" main-color="#0ea5e9">
  <lux-heading level="1">Welcome</lux-heading>
  <lux-card variant="elevated">
    <lux-text>Card content</lux-text>
    <lux-button variant="primary">Action</lux-button>
  </lux-card>
</lux-scope>
```
